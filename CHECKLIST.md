## コードベースで確認した全機能

### 認証
- **ログイン**  
  - Supabase 利用時: `signInWithPassword` → profiles 取得 → 承認済みなら会員/管理へ  
  - 未設定時: `initU`（admin / demo）で照合
- **新規登録**  
  - メール重複チェック → Supabase またはメモリに登録 → 紹介コードがあれば refs/comm 加算 → ウェルカムメール送信（`/api/send-email`）
- **ログアウト**  
  - Supabase の場合は `signOut` → ランディングへ

### 会員画面（MemberApp）
| 機能 | コンポーネント | 内容 |
|------|----------------|------|
| ダッシュボード | PHome | 入力進捗・フォーム/AI対話への導線・各機能カード |
| フォーム入力で作成 | PForm | SEC セクション別入力・保存（upU で fd 更新）・前へ/保存して次へ |
| AI対話で作成 | PChat | `/api/chat` で申請情報引き出し・会話履歴表示 |
| 申請書類をAI生成 | PGen | 入力進捗表示・5項目以上で生成可能・`/api/chat` でドラフト生成・コピー |
| 書類チェックリスト | PCheck | 必要書類のチェック（地域別必須あり） |
| 申請タイムライン | PTL | 1〜5ヶ月目のステップ表示 |
| 補助金情報 | PSub | ものでこり・持続化・政金公庫・信用保証・固定資産税減免 |
| AIアドバイザー | PAdv | `/api/chat` で地域特化の相談 |
| 紹介報酬 | PAff | 紹介人数・累計報酬・単価・紹介URL・コピー |
| チケット購入 | PTkt | BASE 購入リンク |

  - PChat（AI対話で作成）・PGen（申請書類生成）・PAdv（AIアドバイザー）で使用。
